# ElectroFabi IPTV

Plataforma OTT completa con Backend Laravel y App Flutter.

## Características principales
- Transmisión de canales en vivo de baja latencia.
- Soporte para Series y Películas (VOD).
- Sistema de actualización automática.
- Control parental y gestión de perfiles.
